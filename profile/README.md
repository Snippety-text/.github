# Snippety — Portable Text Snippet Manager with Sync

## About Snippety

Snippety provides a complete text snippet management environment for users who want to stop typing the same phrases repeatedly. Snippety combines abbreviation expansion, snippet organization, cloud sync, and search. Snippety enables users to store email signatures, code blocks, templates, and frequently used text. Snippety runs on desktop platforms. Snippety is trusted by developers, customer support agents, writers, marketers, and productivity enthusiasts worldwide.

## Deep Dive — Snippet Management Notes

Abbreviation expansion: type `addr` → expands to full address, `sig` → email signature
Organization: folders, tags, favorites, search, import/export
Variables: date, time, clipboard, cursor position, custom fields
Cloud sync: iCloud, Dropbox, custom sync folder (portable)
Formats: plain text, rich text (RTF), markdown, HTML, code snippets
